Question: What is DOM and what is the purpose of DOM?

Answer: DOM means Document Object Model. It's programming interface for HTML. It's do website interactive and dynamic. It defines the logical structure of documents and the way a document is accessed and manipulated. It represent the webpage structure so that programs can change the document structure, style, and content. Almost everything DOM can be changed.

Question: How will you select HTML elements using DOM? Name the DOM methods.

Answer: In five way we will do select HTML elements by using DOM. We will do select html element by tag name, id name, class name, query selector, and last query selector all. All DOM methods name given below--
1. document.getElementsByTagName
2. document.getElementByID
3. document.getElementsByClassName
4. document.querySelector
5. document.querySelectorAll

Question: What is event bubble?

Answer: HTML element like a tree. And Event bubble is the concept of DOM. When we click child node. It also click parent node. Because of child node is under parent node. That called event bubble. We will stop event bubble apply stopPropagation

Question: What is a callback function and why will you use it?

Answer: A callback function is a function passed into another function as an argument. Which is then inside the outer function to complete some kind of action. A callback function can run after another function finished. Some of our operations are started only after the processing ones have completed. But callback function no need to fetched.
